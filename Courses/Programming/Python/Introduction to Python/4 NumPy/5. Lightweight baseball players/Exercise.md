**Lightweight baseball players**

To subset both regular Python lists and `numpy` arrays, you can use square brackets:

```
x = [4 , 9 , 6, 3, 1]
x[1]
import numpy as np
y = np.array(x)
y[1]
```  

For `numpy` specifically, you can also use boolean `numpy` arrays:

```
high = y > 5
y[high]
```
